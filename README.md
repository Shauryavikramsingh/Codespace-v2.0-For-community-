**A Professional, Full-Featured Browser-Based Code Editor**

CodeSpace is a sophisticated, lightweight IDE-style web application designed for developers who need a clean, high-performance workspace. This project has been bundled into a standalone Python-powered executable to ensure a professional desktop experience and to protect the integrity of the source assets.

---

##  Key Features

###  Professional IDE Interface
- **Activity Bar:** Quick access to File Explorer, Global Search, and Document Outline.
- **Multi-Tab System:** Manage multiple files simultaneously with a native-feeling tab bar.
- **Status Bar:** Real-time updates on cursor position (Line/Col), encoding (UTF-8), and language mode.

###  Advanced Editor Capabilities
- **Mini-map:** A visual overview of your code for fast navigation.
- **Custom Themes:** Includes high-contrast modes: Dark (Default), Light, Dracula, Monokai, and Nord.
- **Find & Replace:** Full search and replace functionality with a dedicated UI bar.
- **Smart Editing:** Auto-save, auto-closing brackets, and adjustable font/tab sizing.

###  Asset Protection & Portability
- **Decentralized Execution:** Wrapped in a Python binary to function as a standalone desktop app.
- **Source Integrity:** Assets like the core `index.html` and UI logic are bundled within the `.exe` to prevent unauthorized remixing.

##  Getting Started

### For Users (Download the App)
You do not need to install any programming languages to use CodeSpace.
1. Navigate to the **[Releases](https://github.com/Shauryavikramsingh/releases)** section.
2. Download `CodeSpace.exe`.
3. Run the file directly.

### Keyboard Shortcuts
| Command | Action |

| `Ctrl + F` | Find & Replace |
| `Ctrl + N` | New File |
| `Alt + Z` | Toggle Word Wrap |
